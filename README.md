# Personal Money Manager

A PHP MySQL web application for managing personal finances, adding expenses, tracking activity, and viewing dashboard summaries.

## Features
- User registration and login
- Add and track expenses
- Dashboard for financial overview
- Separate includes for modular PHP structure
- Clean UI with simple styling
- MySQL database driven storage

## Database
Name: `ems_db`  
Schema file: `database.sql`

## File Structure Overview
- `assets/`
  - `css/style.css`
  - `img/favicon.ico`, `logo.png`
  - `js/script.js`
- `includes/`
  - `db.php` (database connection)
  - `functions.php` (logic functions)
  - `header.php`, `footer.php`
- `index.php` (login page)
- `register.php` (user signup)
- `dashboard.php` (main overview)
- `add_expense.php` (expense input)
- `logout.php`
- `database.sql`

## Installation
1. Import `database.sql` into MySQL  
2. Update DB credentials in `includes/db.php`  
3. Deploy on a PHP supported host or local server
